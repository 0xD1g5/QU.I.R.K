# Iconography — unicode → Lucide swap table

The v1.1 `ds-components.html` reference used unicode glyphs as inline placeholders. The UI kits and any new work should use **Lucide** instead. Mapping below.

| In v1.1 source | Meaning | Lucide name |
|---|---|---|
| `⊞` | Dashboard | `layout-dashboard` |
| `⚠` | Detections / alert | `alert-triangle` |
| `◈` | Assets | `database` (or `server`) |
| `⊟` | Policies | `clipboard-list` |
| `≡` | Logs / list | `list` |
| `⚙` | Settings | `settings` |
| `⌕` | Search | `search` |
| `↕` | Sort (both directions) | `arrows-up-down` |
| `↑` | Up / increase | `arrow-up` |
| `↓` | Down / decrease | `arrow-down` |
| `●` | Status dot | (keep — it's the badge dot) |
