01 / FOUNDATION

Typography

System-ui for all UI and body text. JetBrains Mono for data values, metrics, and code. Labels use 10px + uppercase + letter-spacing — no separate display font needed.

Scale

label / 10px / .08emAll-caps eyebrow label
ui / 12pxTable cell · chip · button · UI chrome
body / 13pxBody text and descriptions for cards and panels
h3 / 15px / 600Card title or sub-section header
h2 / 18px / 600Section title
h1 / 22px / 700Page title
metric / 30px / 700 / mono1,204

01 / FOUNDATION

Spacing

8px base grid. All padding, gap, and margin values are multiples: 4, 8, 12, 16, 20, 24, 32, 40px.

--ds-s8 · 32px · main content padding
--ds-s10 · 40px · between sections
