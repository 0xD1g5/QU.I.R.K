```python
# quirk/util/targets.py
def parse_target_tokens(raw: str) -> tuple[list[str], list[str]]:
    """Routes per token. Returns (fqdns_or_ips, cidrs).

    Per-token routing (D-01):
      - starts with '@' → recursively load file via load_targets_file (file contents
        themselves are NOT re-routed through @-prefix; D-02 forbids nested @file)
      - contains '/'   → validate via ipaddress.ip_network(token, strict=False);
                         on success append to cidrs; on ValueError re-raise with token
      - else           → append to fqdns_or_ips (single bare host / IP)

    Splits on ',' first, then routes each trimmed non-empty token.
    Whitespace tokens are skipped silently.
    """

def load_targets_file(path: str) -> str:
    """Read file; strip blank lines and lines starting with '#'.
    Return the surviving lines joined by ','.
    Raises FileNotFoundError with the path embedded in the message (D-05).
    """
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user CLI input → parser | Untrusted strings (CSV, paths, CIDRs) cross into `parse_target_tokens` / `load_targets_file` |
| `--targets-file` path → filesystem read | User supplies an arbitrary path; CLI-trusted boundary (operator's own machine) |
| Parsed targets → `subprocess.run(nmap, ...)` (downstream, Plan 47-02) | List-form args only; no shell=True |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-47-01 | Tampering | parse_target_tokens | mitigate | Validate CIDR via stdlib `ipaddress.ip_network`; re-raise `ValueError` with offending token (D-05); never silently coerce |
| T-47-02 | Information Disclosure | load_targets_file | accept | CLI-trusted boundary (operator's own targets file). No path-traversal sanitization needed. D-02 forbids nested `@file` to bound the read surface to one file per invocation. |
| T-47-03 | Denial of Service | parse_target_tokens with massive CSV | accept | Parser is O(n) string scan; no recursion beyond the single nested-file expansion. Probe-budget guard in Plan 47-02 catches downstream DoS via large CIDRs. |
| T-47-04 | Tampering | wizard input → fqdns/cidrs assignment | mitigate | All ValueError/FileNotFoundError surface to user before any scan dispatch; no half-populated TargetsCfg. |
</threat_model>
