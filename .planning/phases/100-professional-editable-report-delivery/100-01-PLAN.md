## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-100-LOGO | Information Disclosure / DoS | _load_logo_b64 in html_renderer.py | mitigate | Read only the configured path with `open(logo_path, "rb")` guarded by try/except (OSError, IOError) → return (None, "png") graceful omit (D-03); rendered HTML consumed by Chromium with JS disabled + offline, so a data: URI image is inert |
| T-100-XSS | Tampering | cover-page config strings in report.html.j2 | mitigate | org_name/report_owner/data_classification rendered through the existing `| sanitize` filter; generated_at is datetime-derived, not user input |
| T-100-IMG | Tampering | logo data: URI img src | accept | logo_b64 is base64 of raw bytes; no HTML is rendered from the path; mime subtype is a fixed allow-list lookup defaulting to "png" |
</threat_model>
