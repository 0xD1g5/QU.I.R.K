2. **`PLATFORM_VERSION = "3.9"`** — import from `quirk.reports.writer` or duplicate to avoid circular import.

3. **`_CIPHER_SUITE_DECOMPOSITION: dict[str, list[str]]`** — A mapping of common TLS cipher suite fragments to CycloneDX canonical algorithm names. Use a decomposition function rather than a static map:

   ```python
   def _decompose_cipher_suite(suite: str) -> list[str]:
       """Decompose a TLS cipher suite name into constituent algorithm names.
       E.g. 'TLS_ECDHE_RSA_WITH_AES_256_GCM_SHA384' ->
       ['X25519', 'RSA', 'AES-256-GCM', 'SHA-384']
       """
   ```

   Logic: Split on `_` and `WITH`, match fragments:
   - `ECDHE` -> `"X25519"` (generic ECDHE, actual curve unknown from suite name)
   - `RSA` -> `"RSA"` (when in key-exchange/auth position)
   - `AES_256_GCM` or `AES_128_GCM` -> `"AES-256-GCM"` / `"AES-128-GCM"`
   - `AES_256_CBC` -> `"AES-256-CBC"`
   - `CHACHA20_POLY1305` -> `"ChaCha20-Poly1305"`
   - `SHA384` / `SHA256` / `SHA` -> `"SHA-384"` / `"SHA-256"` / `"SHA-1"`
   - `3DES_EDE_CBC` -> `"3DES"`

   For TLSv1.3 suites (e.g., `TLS_AES_256_GCM_SHA384`): no key exchange component (it's in the handshake, not the suite name). Decompose only the cipher + hash.

4. **`_make_algorithm_component(name: str, bom_ref_key: str, key_size: int | None = None) -> Component`**:
   - Call `classify_algorithm(name)` to get `(primitive, nist_level, classical_level)`
   - Construct `Component(type=CRYPTOGRAPHIC_ASSET, bom_ref=bom_ref_key, name=display_name, crypto_properties=CryptoProperties(asset_type=ALGORITHM, algorithm_properties=AlgorithmProperties(primitive=primitive, nist_quantum_security_level=nist_level, classical_security_level=classical_level, execution_environment=SOFTWARE_PLAIN_RAM)))`
   - If `key_size` provided, set `parameter_set_identifier=str(key_size)`
   - Use bom_ref format: `"crypto/algorithm/{normalized_name}"`

5. **`_extract_ssh_algorithms(ssh_audit_json_str: str) -> dict[str, list[dict]]`**:
   - Parse JSON, return `{"kex": [...], "key": [...], "enc": [...], "mac": [...]}`
   - Handle None / invalid JSON gracefully (return empty dict)

6. **`build_cbom(endpoints: list[CryptoEndpoint]) -> Bom`**:
   Processing order (per research pitfall #3 — algorithms first, then certs, then protocols):

   a. **Pass 1 — Collect algorithm components** into `algo_registry: dict[str, Component]` keyed by bom_ref string.

      For each endpoint:
      - If TLS (protocol != "SSH"): decompose `cipher_suite` via `_decompose_cipher_suite()`. For each algorithm name, create/reuse component in `algo_registry`.
      - If cert_pubkey_alg is set: create algorithm component named `"{cert_pubkey_alg}-{cert_pubkey_size}"` (e.g., "RSA-2048"). Register in `algo_registry`.
      - If SSH (protocol == "SSH" and ssh_audit_json is not None): parse ssh_audit_json, iterate kex/key/enc/mac arrays, create/reuse algorithm component for each.

   b. **Pass 2 — Build certificate components** (only for TLS endpoints with cert_pubkey_alg):
      ```python
      CryptoProperties(
          asset_type=CryptoAssetType.CERTIFICATE,
          certificate_properties=CertificateProperties(
              subject_name=ep.cert_subject,
              issuer_name=ep.cert_issuer,
              not_valid_before=ep.cert_not_before,
              not_valid_after=ep.cert_not_after,
              signature_algorithm_ref=BomRef(value=sig_alg_bom_ref),
              subject_public_key_ref=BomRef(value=pubkey_bom_ref),
              certificate_format="X.509",
          ),
      )
      ```
      Use bom_ref format: `"crypto/certificate/{host}:{port}"`

   c. **Pass 3 — Build protocol components**:
      - TLS endpoints: `ProtocolPropertiesType.TLS`, version from `ep.tls_version` (strip "TLSv" prefix), cipher_suites list with BomRef references to algorithm components.
      - SSH endpoints: `ProtocolPropertiesType.SSH`, version "2.0", cipher_suites referencing KEX algorithm bom_refs.
      Use bom_ref format: `"crypto/protocol/{tls|ssh}/{host}:{port}"`

   d. **Assemble Bom:**
      ```python
      bom = Bom(
          components=list(algo_registry.values()) + cert_components + protocol_components,
          metadata=BomMetaData(
              timestamp=datetime.now(timezone.utc),
              component=Component(
                  name="QU.I.R.K.",
                  type=ComponentType.APPLICATION,
                  version=PLATFORM_VERSION,
              ),
          ),
      )
      ```
