**docs/installation.md** — Full reference for system requirements and platform-specific steps.

Structure:
1. **System Requirements**
   - Python 3.10 or higher (`python3 --version` to check)
   - pip 21.0+
   - git
   - Docker Desktop (optional — required only for chaos lab)
   - Playwright-compatible OS for PDF export (macOS 10.15+, Ubuntu 20.04+, Windows 10 via WSL2)

2. **macOS**
   - Homebrew Python recommended: `brew install python@3.12`
   - Full install sequence (same as Getting Started)
   - Playwright note: `playwright install chromium` installs to `~/.local/share/ms-playwright/`
   - Common issue: if `quirk` not found after install, confirm venv is activated

3. **Linux (Ubuntu/Debian)**
   - System packages: `sudo apt-get update && sudo apt-get install -y python3 python3-pip python3-venv git`
   - Full install sequence
   - Playwright dependencies: `playwright install-deps chromium` (installs system libs)
   - Note: Playwright requires glibc 2.17+ (Ubuntu 20.04+)

4. **Windows (WSL2)** — separate section per D-07
   - Enable WSL2: `wsl --install`
   - Recommended distro: Ubuntu 22.04
   - After WSL2 setup, follow the Linux instructions above
   - Dashboard: `quirk serve` starts on `127.0.0.1:8512` — access from Windows browser at `http://localhost:8512`
   - Note: Docker Desktop must have WSL2 integration enabled for chaos lab use

5. **Optional Dependencies**
   - `pip install -e '.[dashboard]'` — FastAPI + Playwright for web dashboard and PDF export
   - `pip install syft` — required for container scanning
   - `pip install semgrep` — required for source code scanning
   - `pip install 'quirk[dashboard]' boto3 azure-identity azure-keyvault-keys azure-mgmt-network` — full install for cloud connectors

6. **Verify Installation**
