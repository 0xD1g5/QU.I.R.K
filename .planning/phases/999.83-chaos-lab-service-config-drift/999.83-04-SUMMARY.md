# Plan 999.83-04 SUMMARY — MySQL pin to 8.0 (BACK-90 bug 4)

**Commit:** see below

## Change

`quantum-chaos-enterprise-lab/docker-compose.yml` — single line, mysql-ssl-off service:

```diff
   mysql-ssl-off:
-    image: mysql:8
+    image: mysql:8.0
```

Per CONTEXT D-4 option (a): the floating `mysql:8` tag now resolves to MySQL 8.4.x which removed the `--skip-ssl` flag. Pinning to `mysql:8.0` (last major-version supporting the flag) preserves the lab's documented insecure-by-default behavior and is a closer match to enterprise MySQL deployments in 2026.
