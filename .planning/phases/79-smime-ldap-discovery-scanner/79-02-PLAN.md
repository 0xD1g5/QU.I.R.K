ldap3 search pattern (D-79-R4 â€” search uses raw_attributes, NOT `;binary`):
```python
server = ldap3.Server(host, port=port, get_info=ldap3.ALL, connect_timeout=timeout)
conn = ldap3.Connection(server, authentication=ldap3.ANONYMOUS, receive_timeout=timeout)
if not conn.bind(): return []
entries = conn.extend.standard.paged_search(
    search_base=base_dn,
    search_filter='(|(userCertificate=*)(userSMIMECertificate=*))',
    search_scope=ldap3.SUBTREE,
    attributes=['userCertificate', 'userSMIMECertificate', 'cn', 'uid'],
    paged_size=500,
    generator=True,
)
for entry in entries:
    raw = entry.get('raw_attributes', {})
    for cert_bytes in raw.get('userSMIMECertificate', []):
        ...  # cert_bytes is bytes (DER)
```

saml_scanner.py pattern for DER parse + severity classifier:
```python
from cryptography.x509 import load_der_x509_certificate
cert = load_der_x509_certificate(der_bytes)
pub = cert.public_key()
if isinstance(pub, rsa.RSAPublicKey):
    key_alg, key_bits = "RSA", pub.key_size
elif isinstance(pub, ec.EllipticCurvePublicKey):
    key_alg, key_bits = "ECDSA", pub.key_size
sig_hash = cert.signature_hash_algorithm.name  # 'sha1', 'sha256', ...
```
