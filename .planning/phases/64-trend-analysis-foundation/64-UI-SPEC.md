## Component Inventory

### New Components

#### `RegressionAlertChip`
- Location: `src/dashboard/src/components/RegressionAlertChip.tsx`
- Renders: dismissible alert chip on `ExecutivePage` above the score gauge section
- Structure: single `<div>` using `ds-critical-bdr` border + `ds-critical-dim` background, flex row, warning icon (Lucide `AlertTriangle` 16px), message text (14px/400), "View trends" anchor (accent), dismiss "×" button (ghost, 16px icon)
- Visibility: hidden when `localStorage.getItem("quirk.dismissed_regression.<session_ts>")` is truthy or no regression detected
- Dismiss interaction: click "×" → `localStorage.setItem(key, "1")` → `useState` hides chip immediately, no page reload
- Regression condition: `score_delta !== null && score_delta <= -5` OR `new_high > 0` (from `useTrendsData()`)

#### `useTimelineData` hook
- Location: `src/dashboard/src/hooks/useTimelineData.ts`
- Fetch target: `GET /api/trends/timeline?n=30`
- Pattern: cancellation-safe (`let cancelled = false` inside `useEffect`), mirrors `useTrendsData.ts` exactly
- Returns: `{ data: TrendTimeline | null, loading: boolean, error: string | null }`

### Modified Components

#### `TrendsPage` (`src/dashboard/src/pages/trends.tsx`)
- Add `<ChartContainer>` + `<LineChart>` with 7 `<Line>` series ABOVE the existing Score Delta card
- Section heading for chart: `"Score & Pillar Timeline"` at 14px/semibold
- Chart height: aspect-video (ChartContainer default), min-height implicitly ~300px via 16:9 ratio
- All 7 `<Line>` components always mounted; toggle via `strokeOpacity` only (never conditional render)
- XAxis: `dataKey="session_ts"`, tickFormatter converts ISO → `MM/DD HH:mm` using `new Date(v).toLocaleString([], {month:'2-digit', day:'2-digit', hour:'2-digit', minute:'2-digit'})`
- YAxis: `domain={[0, 100]}`, tickCount 6 (0, 20, 40, 60, 80, 100)
- Tooltip: custom — shows `new Date(session_ts).toLocaleString()` as header + 7 labeled score rows
- Loading state: `<PageSpinner ariaLabel="Loading trend timeline" />` (replaces chart area only)
- Empty state (< 2 sessions): existing copy "No scan history yet. Run two or more scans to see trend lines." — no chart rendered

#### `ExecutivePage` (`src/dashboard/src/pages/executive.tsx`)
- Insert `<RegressionAlertChip />` immediately before the score gauge section
- No layout changes to the gauge or existing cards

## Interaction Contract

### Chart Hover Tooltip
- Trigger: mouse enter on any data point or chart area
- Response time: immediate (Recharts built-in, no async)
- Content: timestamp header (full locale string) + 7 score rows with color-matched labels
- Dismiss: mouse leave

### RegressionAlertChip Dismiss
- Trigger: click "×" button
- Action: `localStorage.setItem("quirk.dismissed_regression.<session_ts>", "1")`
- Visual feedback: chip unmounts immediately (React state, no animation required)
- Persistence: survives page refresh; cleared automatically when `session_ts` advances (new scan)
- No confirmation dialog required — dismissal is not destructive

### Deep-link from Chip
- Element: anchor `<a href="/trends">View trends →</a>` (React Router `<Link to="/trends">`)
- Styled with accent color `hsl(var(--primary))`
- Navigates to `/trends` — no new tab
