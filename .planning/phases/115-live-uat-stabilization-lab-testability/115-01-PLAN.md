## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-115-01 | Information Disclosure | console_cmd._cmd_enroll idempotent path | mitigate | Idempotent path prints NO raw token (D-01); pre-check precedes secrets.token_urlsafe so no stale token can leak (Pitfall 1) |
| T-115-02 | Information Disclosure | enroll error/info messages | mitigate | T-109 fixed-string convention; never stringify exceptions in the "already enrolled" notice |
| T-115-03 | DoS | duplicate enroll flooding | mitigate | Pre-check + retained IntegrityError backstop prevent repeated inserts and token churn |
| T-115-04 | Tampering | advisory-row injection into console via push | mitigate | STAB-04 filter excludes missing_extra rows at the push/export boundary; NULL clause prevents accidental drop of real findings |
| T-115-05 | Elevation/Integrity | one-time-token guarantee | accept | Idempotent enroll mints no new token, so the one-time guarantee is strengthened, not weakened; --rotate deferred (D-03) |
</threat_model>
