**Implement each helper function:**

1. `_fetch_metadata(url: str, timeout: int) -> bytes | None`:
   - Use `httpx.get(url, timeout=timeout, follow_redirects=True, verify=False)` per D-13, D-14
   - httpx follow_redirects handles up to the default redirect limit
   - Return `response.content` (bytes) on HTTP 200
   - Return `None` on any exception or non-200 status
   - Log warning on failure

2. `_classify_target(url: str, content: bytes) -> str`:
   - If `".well-known"` in url, return `"oidc"` per D-01
   - Try `json.loads(content)` — if succeeds, return `"oidc"`
   - Otherwise return `"saml"`

3. `_parse_cert_element(cert_b64_text: str) -> dict | None`:
   - Strip ALL whitespace: `cert_b64_text.replace(" ", "").replace("\n", "").replace("\r", "").strip()` per Pitfall 7
   - `der = base64.b64decode(cleaned)`
   - `cert = load_der_x509_certificate(der)`
   - `pub = cert.public_key()`
   - If `isinstance(pub, rsa.RSAPublicKey)`: `key_alg="RSA"`, `key_bits=pub.key_size`
   - If `isinstance(pub, ec.EllipticCurvePublicKey)`: `key_alg="ECDSA"`, `key_bits=pub.key_size`
   - Else: `key_alg="UNKNOWN"`, `key_bits=None`
   - Return `{"key_alg": key_alg, "key_bits": key_bits, "serial": format(cert.serial_number, 'x'), "not_after": cert.not_valid_after_utc.isoformat()}`
   - Return `None` on any exception (invalid cert data)

4. `_is_sha1_uri(uri: str) -> bool`:
   - `lower = uri.lower()`
   - `return any(ind in lower for ind in SHA1_INDICATORS)`

5. `_classify_key_severity(key_alg: str, key_bits: int | None) -> str | None`:
   - Per D-07:
     - If `key_alg == "RSA"` and `key_bits is not None`:
       - `key_bits < 2048` -> `"CRITICAL"`
       - `key_bits == 2048` -> `"HIGH"`
       - `key_bits > 2048` -> `"HIGH"` (still quantum-vulnerable RSA)
     - If `key_alg in ("ECDSA", "EdDSA", "Ed25519")`: return `None` (SAFE)
     - Else: return `None`

6. `_parse_saml_metadata(xml_bytes: bytes, target_url: str) -> tuple[list, dict]`:
   - `root = defused_ET.fromstring(xml_bytes)` — use defusedxml to prevent XXE per D-11
   - Extract entityID: `entity_id = root.get("entityID", "")` — also check for EntitiesDescriptor wrapper: `ed = root.find("md:EntityDescriptor", namespaces=SAML_NS); if ed is not None: entity_id = ed.get("entityID", "")`
   - Parse port from target_url: `parsed = urlparse(target_url); port = parsed.port or (443 if parsed.scheme == "https" else 80)`
   - Initialize `scan_dict = {"url": target_url, "type": "saml", "entity_id": entity_id, "signing_certs": [], "encryption_certs": [], "sha1_uris": [], "errors": []}` per D-17
   - Initialize `endpoints = []`

   **Signing certs (SAML-01):**
   - XPath: `root.findall(".//md:KeyDescriptor[@use='signing']/ds:KeyInfo/ds:X509Data/ds:X509Certificate", namespaces=SAML_NS)`
   - Also handle KeyDescriptor WITHOUT `use` attribute (default is both signing and encryption — treat as signing): `root.findall(".//md:KeyDescriptor[not(@use)]/ds:KeyInfo/ds:X509Data/ds:X509Certificate", namespaces=SAML_NS)`
   - For each cert element: `cert_info = _parse_cert_element(elem.text)`. If None, skip.
   - Append to `scan_dict["signing_certs"]`: `{"serial": cert_info["serial"], "key_alg": cert_info["key_alg"], "key_bits": cert_info["key_bits"], "not_after": cert_info["not_after"]}`
   - Create CryptoEndpoint per D-02:
     ```python
     ep = CryptoEndpoint(
         host=parsed.hostname or target_url,
         port=port,
         protocol="SAML",
         cert_pubkey_alg=cert_info["key_alg"],
         cert_pubkey_size=cert_info["key_bits"],
         service_detail=f"{entity_id}|use=signing|serial={cert_info['serial']}",
         saml_scan_json=json.dumps(scan_dict),
         scanned_at=datetime.now(timezone.utc).replace(tzinfo=None),
     )
     endpoints.append(ep)
     ```

   **Encryption certs (SAML-02):**
   - XPath: `root.findall(".//md:KeyDescriptor[@use='encryption']/ds:KeyInfo/ds:X509Data/ds:X509Certificate", namespaces=SAML_NS)`
   - Same cert parsing and CryptoEndpoint construction, but `service_detail` uses `use=encryption`
   - Append to `scan_dict["encryption_certs"]`

   **SHA-1 URI detection (D-05):**
   - SignatureMethod: `root.findall(".//ds:SignatureMethod", namespaces=SAML_NS)` — for each, check `_is_sha1_uri(elem.get("Algorithm", ""))`
   - SigningMethod: `root.findall(".//alg:SigningMethod", namespaces=SAML_NS)` — same check
   - For each SHA-1 hit, append to `scan_dict["sha1_uris"]`: `{"source": source_name, "uri": uri}`
   - Create CryptoEndpoint per D-04:
     ```python
     ep = CryptoEndpoint(
         host=parsed.hostname or target_url,
         port=port,
         protocol="SAML",
         cert_pubkey_alg="SHA1",
         cert_pubkey_size=None,
         service_detail=f"{entity_id}|algo_uri={uri}|source={source_name}",
         saml_scan_json=json.dumps(scan_dict),
         scanned_at=datetime.now(timezone.utc).replace(tzinfo=None),
     )
     endpoints.append(ep)
     ```

   - Return `(endpoints, scan_dict)`

7. `_parse_oidc_discovery(json_bytes: bytes, target_url: str) -> tuple[list, dict]`:
   - `data = json.loads(json_bytes)` per D-03
   - `issuer = data.get("issuer", target_url)`
   - `parsed = urlparse(target_url); port = parsed.port or (443 if parsed.scheme == "https" else 80)`
   - `scan_dict = {"url": target_url, "type": "oidc", "issuer": issuer, "id_token_signing_alg_values_supported": [], "request_object_signing_alg_values_supported": [], "errors": []}` per D-17
   - `endpoints = []`
   - Iterate `data.get("id_token_signing_alg_values_supported", [])` per Pitfall 6
   - For each alg string: check `OIDC_ALG_SEVERITY.get(alg)`. If severity is not None (i.e., RSA algs per D-09):
     ```python
     ep = CryptoEndpoint(
         host=parsed.hostname or target_url,
         port=port,
         protocol="SAML",  # Still "SAML" protocol — OIDC findings stored under SAML umbrella
         cert_pubkey_alg=alg,
         cert_pubkey_size=2048 if alg.startswith("RS") or alg.startswith("PS") else None,
         service_detail=f"oidc-discovery|id_token_signing_alg",
         saml_scan_json=json.dumps(scan_dict),
         scanned_at=datetime.now(timezone.utc).replace(tzinfo=None),
     )
     endpoints.append(ep)
     ```
   - Also iterate `data.get("request_object_signing_alg_values_supported", [])` — same logic, `service_detail=f"oidc-discovery|request_object_signing_alg"`
   - Populate `scan_dict` lists with the alg arrays from data
   - Return `(endpoints, scan_dict)`

8. `scan_saml_targets(targets: list, timeout: int = 10, logger=None) -> list`:
   - If `not LXML_AVAILABLE`: log warning, return `[]`
   - `log = logger or logging.getLogger(__name__)`
   - `all_endpoints = []`
   - For each `target_url` in targets (sequential per D-12):
     - `content = _fetch_metadata(target_url, timeout)` — if None, log warning, continue
     - `target_type = _classify_target(target_url, content)` per D-01
     - If `target_type == "oidc"`:
       - `endpoints, scan_dict = _parse_oidc_discovery(content, target_url)`

       - On any exception: log warning, add error to scan_dict, skip
     - `all_endpoints.extend(endpoints)`
   - Return `all_endpoints`

All error handling: log warning, skip target, continue (D-15). Never raise exceptions to caller.
