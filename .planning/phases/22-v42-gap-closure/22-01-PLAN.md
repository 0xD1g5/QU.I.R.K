**Fix 2 — builder.py Pass 2: Add SAML and KERBEROS to certificate skip list (line 389)**

Change line 389 from:
```python
if ep.protocol in ("SSH", "CONTAINER", "SOURCE"):
```
to:
```python
if ep.protocol in ("SSH", "CONTAINER", "SOURCE", "KERBEROS", "SAML"):
```

Rationale: Neither the Kerberos scanner nor the SAML scanner sets `cert_subject`, `cert_issuer`, `cert_sig_alg`, `cert_not_before`, or `cert_not_after`. Kerberos endpoints store etype names (like "rc4-hmac") in `cert_pubkey_alg` — these are NOT certificate algorithms. SAML endpoints store key_alg or "SHA1" — they do NOT have X.509 cert metadata in these fields. Allowing them through Pass 2 creates hollow certificate components with empty subject/issuer/dates.

**Fix 3 — builder.py Pass 3: Add SAML and KERBEROS to protocol skip list (line 468)**

Change line 468 from:
```python
elif ep.protocol in ("JWT", "CONTAINER", "SOURCE", "AWS", "AZURE", "DNSSEC"):
```
to:
```python
elif ep.protocol in ("JWT", "CONTAINER", "SOURCE", "AWS", "AZURE", "DNSSEC", "SAML", "KERBEROS"):
```

Rationale: SAML and Kerberos are not TLS/SSH network protocols. Their cryptographic findings are already captured in Pass 1 (algorithm components). Without this fix, they fall through to the `else` TLS block and get spurious `crypto/protocol/tls/{host}:{port}` components.

# SAML protocol tests (Phase 22 gap closure — SAML-05)

def test_saml_endpoint_algorithm_registered():
    """SAML endpoint registers algorithm component from cert_pubkey_alg."""
    ep = _saml_endpoint(cert_pubkey_alg="RSA", cert_pubkey_size=2048)
    bom = build_cbom([ep])
    algo_refs = [c.bom_ref for c in bom.components if "crypto/algorithm/" in str(c.bom_ref)]
    assert any("rsa" in str(ref) for ref in algo_refs), f"RSA algorithm not found in {algo_refs}"


def test_saml_endpoint_no_tls_protocol():
    """SAML endpoint must NOT produce a TLS protocol component."""
    ep = _saml_endpoint()
    bom = build_cbom([ep])
    tls_protos = [c for c in bom.components if str(c.bom_ref).startswith("crypto/protocol/tls/")]
    assert tls_protos == [], f"Spurious TLS protocol components for SAML: {[str(c.bom_ref) for c in tls_protos]}"


def test_saml_endpoint_no_certificate():
    """SAML SHA1 finding must NOT produce a certificate component (no cert metadata)."""
    ep = _saml_endpoint(cert_pubkey_alg="SHA1", cert_pubkey_size=None)
    bom = build_cbom([ep])
    cert_comps = [c for c in bom.components if str(c.bom_ref).startswith("crypto/certificate/")]
    assert cert_comps == [], f"Spurious certificate components for SAML: {[str(c.bom_ref) for c in cert_comps]}"
