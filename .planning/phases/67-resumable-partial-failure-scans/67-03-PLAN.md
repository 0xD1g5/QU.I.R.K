use _wrapped_phase() for uniform error capture. This ensures every scanner
failure produces a consistent error_endpoints row with scan_error_category set,
which in turn feeds the partial_failures array built in Plan 02.

results instead of producing partial_failures entries. The _wrapped_phase
pattern already exists for TLS, SSH, broker — this plan brings all remaining
scanners into line.

invocations use _wrapped_phase() or (for scanners that only run conditionally
on a flag check + no external crash risk) remain as direct calls inside a
_wrapped_phase lambda. The email scanner's existing BaseException try/except is
replaced by _wrapped_phase.

```python
def _wrapped_phase(run_stats, phase_name, scanner_label, fn, error_endpoints, logger):
    """Runs fn() under BaseException protection. Returns [] on non-fatal exception."""
    try:
        with _phase_timer(run_stats, phase_name):
            return fn()
    except (KeyboardInterrupt, SystemExit):
        raise
    except BaseException as exc:
        logger.error(f"{phase_name}: unhandled exception: {exc!r}")
        error_endpoints.append(CryptoEndpoint(
            host=scanner_label,
            port=0,
            protocol="ERROR",
            scan_error=str(exc) or exc.__class__.__name__,
            scan_error_category="exception",
        ))
        return []
```
