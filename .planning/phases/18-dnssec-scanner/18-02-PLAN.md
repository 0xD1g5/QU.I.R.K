**`_resolve_ns(domain: str, timeout: int) -> list[str]`** (per D-12):
- Use `dns.resolver.resolve(domain, "NS", lifetime=timeout)` to get NS hostnames
- For each NS hostname, resolve A records via `dns.resolver.resolve(ns_name, "A", lifetime=timeout)`
- Return list of IP address strings
- Catch all exceptions, return `[]` on failure

**`_query_rrset(domain: str, rdtype, ns_ip: str, timeout: int)`** (per D-01, D-13):
- Create query with `dns.message.make_query(domain, rdtype, want_dnssec=True)`
- Clear RD flag: `request.flags &= ~dns.flags.RD` (anti-pattern: authoritative servers refuse RD)
- Call `dns.query.udp_with_fallback(request, ns_ip, timeout=timeout)`
- Return the response object
- On exception, return None

**`_parse_dnskeys(dnskey_rrset) -> list[dict]`** (per Research Pattern 2):
- For each rdata in rrset:
  - `alg_num = int(rdata.algorithm)`
  - `key_tag = dns.dnssec.key_id(rdata)`
  - `role = "KSK" if (rdata.flags & 0x0001) else "ZSK"` (SEP bit)
  - RSA key size (alg in 1,5,7,8,10): parse RFC 3110 format:
    ```python
    key_bytes = rdata.key
    if key_bytes[0] == 0:
        exp_len = (key_bytes[1] << 8) | key_bytes[2]
        modulus_start = 3 + exp_len
    else:
        exp_len = key_bytes[0]
        modulus_start = 1 + exp_len
    key_size = (len(key_bytes) - modulus_start) * 8
    ```
  - Non-RSA: `key_size = None`
- Return list of dicts: `{"flags": int, "alg": int, "tag": int, "key_size": int|None, "role": str}`

**`_parse_ds_records(ds_rrset) -> list[dict]`** (per Research Pattern 3):
- Return `[{"key_tag": rdata.key_tag, "algorithm": int(rdata.algorithm), "digest_type": rdata.digest_type} for rdata in ds_rrset]`

**`_check_chain(dnskeys: list[dict], ds_records: list[dict]) -> bool`** (per Research Pattern 3):
- `dnskey_tags = {d["tag"] for d in dnskeys}`
- `ds_tags = {ds["key_tag"] for ds in ds_records}`
- Return `bool(dnskey_tags & ds_tags)` — True if any key_tag matches

**`_detect_nsec_type(domain: str, ns_ip: str, timeout: int) -> str | None`** (per Research Pattern 4, per D-05):
- Query for `f"_quirk_probe_.{domain}"` A record with DO bit via `_query_rrset`
- Iterate `response.authority` rrsets:
  - If `rrset.rdtype == dns.rdatatype.NSEC` (47): return `"NSEC"`
  - If `rrset.rdtype == dns.rdatatype.NSEC3` (50): return `"NSEC3"`
- Return None on failure or no match

**`_scan_domain(domain: str, timeout: int, logger) -> list[CryptoEndpoint]`**:
Core per-domain logic (per D-01, D-02, D-03, D-11):

1. Resolve NS: `ns_ips = _resolve_ns(domain, timeout)`. If empty, log warning, return `[]`.
2. Pick first NS IP. Query DNSKEY: `dnskey_response = _query_rrset(domain, dns.rdatatype.DNSKEY, ns_ip, timeout)`.
3. If no DNSKEY in answer (unsigned zone — per D-07, DNSSEC-03):
   - Build scan_dict: `{"domain": domain, "ns_queried": ns_ip, "signed": False, "dnskeys": [], "ds_records": [], "nsec_type": None, "chain_valid": None}`
   - Return single CryptoEndpoint: `host=domain, port=53, protocol="DNSSEC", cert_pubkey_alg="NONE", cert_pubkey_size=None, dnssec_scan_json=json.dumps(scan_dict), service_detail="unsigned-zone", scanned_at=datetime.now(timezone.utc).replace(tzinfo=None)`
4. Parse DNSKEY rrset: `dnskeys = _parse_dnskeys(dnskey_rrset)`
5. Detect NSEC type: `nsec_type = _detect_nsec_type(domain, ns_ip, timeout)`
6. Query DS records: `ds_response = _query_rrset(domain, dns.rdatatype.DS, ns_ip, timeout)`. Parse if present: `ds_records = _parse_ds_records(ds_rrset)` or `[]`.
7. Check chain: `chain_valid = _check_chain(dnskeys, ds_records) if ds_records else None`
8. Build scan_dict (per D-03): `{"domain": domain, "ns_queried": ns_ip, "signed": True, "dnskeys": dnskeys, "ds_records": ds_records, "nsec_type": nsec_type, "chain_valid": chain_valid}`
9. Build CryptoEndpoints list:
   - One per DNSKEY (per D-01): For each key in dnskeys, look up `DNSSEC_ALG_MAP.get(key["alg"], (f"UNKNOWN-{key['alg']}", "HIGH"))`. Create CryptoEndpoint with `cert_pubkey_alg=alg_name`, `cert_pubkey_size=key["key_size"]`, `service_detail=f"dnskey:tag={key['tag']}:role={key['role']}"`, `dnssec_scan_json=json.dumps(scan_dict)`.
   - NSEC exposure (per D-02, D-05): If `nsec_type == "NSEC"`, add CryptoEndpoint with `cert_pubkey_alg="NSEC"`, `service_detail="nsec-exposure"`.
   - DS chain broken (per D-02, D-06): If `chain_valid is False`, add CryptoEndpoint with `cert_pubkey_alg="DS-MISMATCH"`, `service_detail="ds-chain-broken"`.
   - DS SHA-1 digest (per D-06): For each DS with `digest_type == 1`, add CryptoEndpoint with `cert_pubkey_alg="SHA1-DS"`, `service_detail="sha1-ds-digest"`.
10. Return endpoints list.
