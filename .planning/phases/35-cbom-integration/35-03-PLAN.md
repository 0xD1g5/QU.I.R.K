bom_ref shapes (from builder.py):
- algorithm: "crypto/algorithm/{normalized_key}" (e.g. "crypto/algorithm/rsa")
- certificate: "crypto/certificate/{host}:{port}"
- protocol (TLS): "crypto/protocol/tls/{host}:{port}"

Normalization for snapshot diff (planner choice):
- Strip metadata (serial number, version, timestamps) — keep only `components`
- For each component: keep `{bom_ref, name, type, crypto_properties.asset_type,
  crypto_properties.protocol_properties.type, crypto_properties.protocol_properties.version,
  cipher_suite_names: [c.name for c in cipher_suites]}` — sorted by bom_ref
