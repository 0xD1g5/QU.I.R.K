<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → FastAPI router | Untrusted JSON payload crosses here (industry, evidence_note, regulatory_obligations) |
| FastAPI → SQLite | Parameterized ORM via SQLAlchemy — no raw SQL with user input in this plan |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-54-01 | Tampering | POST /api/qramm/assessment/draft answer_value | mitigate | Pydantic `Field(ge=1, le=4)` rejects out-of-range values with 422 (test_draft_answer_validation covers) |
| T-54-02 | Tampering | POST /api/qramm/assessment/draft question_number | mitigate | Pydantic `Field(ge=1, le=120)` rejects out-of-range; test_draft_answer_validation covers |
| T-54-03 | Information Disclosure | evidence_note (free text up to 2000 chars) | mitigate | Pydantic `max_length=2000`; React JSX auto-escapes on render (downstream); SQLAlchemy parameterizes inserts |
| T-54-04 | Tampering | POST /api/qramm/profiles regulatory_obligations | mitigate | Stored as `json.dumps(list)`; Pydantic List[str] enforces type; no eval/exec |
| T-54-05 | Denial of Service | List endpoints with no LIMIT | accept | Single-user CLI tool — no auth, no shared multi-tenant; session count bounded by single-consultant workflow |
| T-54-06 | Repudiation | confirmed_at timestamp | mitigate | Server-set via `_now_iso()` — never accepted from client payload |
| T-54-07 | Tampering | ALTER TABLE in migration | mitigate | `_SAFE_COL_RE.match(col)` guard prevents injection via column names; column names are static literals in this code path |
</threat_model>
