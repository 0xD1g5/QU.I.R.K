**Internal scan functions** (each returns `list[CryptoEndpoint]`):
1. `_scan_acm(session, logger)` — Uses `client.get_paginator("list_certificates")`. For each cert summary, calls `describe_certificate` to get `KeyAlgorithm`. Creates CryptoEndpoint: `protocol="AWS"`, `host=CertificateArn`, `port=0`, `cert_pubkey_alg=KeyAlgorithm`, `cloud_scan_json=json.dumps(detail, default=str)`, `service_detail="ACM"`.

2. `_scan_kms(session, logger)` — Uses `client.get_paginator("list_keys")`. For each key, calls `describe_key` to get `KeyMetadata`. Maps `KeySpec` (with fallback to `CustomerMasterKeySpec`) through `KMS_KEY_SPEC_MAP` to get `(alg_name, key_size)`. Creates CryptoEndpoint: `protocol="AWS"`, `host=Arn`, `port=0`, `cert_pubkey_alg=alg_name`, `cert_pubkey_size=key_size`, `cloud_scan_json=json.dumps(metadata, default=str)`, `service_detail="KMS"`.

3. `_scan_cloudfront(session, logger)` — Uses `client.get_paginator("list_distributions")`. For each distribution, extracts `ViewerCertificate.MinimumProtocolVersion`. Creates CryptoEndpoint: `protocol="AWS"`, `host=ARN`, `port=443`, `tls_version=MinimumProtocolVersion`, `cloud_scan_json=json.dumps(dist_data, default=str)`, `service_detail="CloudFront"`.

4. `_scan_elbv2(session, logger)` — Uses `client.get_paginator("describe_load_balancers")` then `describe_listeners` per LB. For HTTPS listeners, extracts `SslPolicy`. Creates CryptoEndpoint: `protocol="AWS"`, `host=ListenerArn`, `port=listener_port`, `tls_version=SslPolicy`, `cloud_scan_json=json.dumps(listener, default=str)`, `service_detail="ELBv2"`.

**Public function:**
```python
def scan_aws_targets(region: str, profile: str | None = None, logger=None) -> list[CryptoEndpoint]:
    if not BOTO3_AVAILABLE:
        if logger:
            logger.v("boto3 not installed — AWS scanning unavailable")
        return []
    session = boto3.Session(region_name=region, profile_name=profile)
    results = []
    results.extend(_scan_acm(session, logger))
    results.extend(_scan_kms(session, logger))
    results.extend(_scan_cloudfront(session, logger))
    results.extend(_scan_elbv2(session, logger))
    return results
```

Each internal function wraps all AWS API calls in try/except, logs errors, returns empty list on failure. No credentials are stored — ambient credential resolution only (per D-15).

**Internal scan functions:**
1. `_scan_keyvault_keys(credential, vault_url, logger)` — Instantiates `KeyClient(vault_url, credential)`. Calls `list_properties_of_keys()`. For each key: extracts `key_type` (string), `key_size`, `name`, `id`. Maps through `AZURE_KEY_TYPE_MAP`. Creates CryptoEndpoint: `protocol="AZURE"`, `host=key.id or vault_url`, `port=0`, `cert_pubkey_alg=mapped_alg`, `cert_pubkey_size=key.key_size`, `cloud_scan_json=json.dumps({"name": key.name, "key_type": str(key.key_type), "key_size": key.key_size, "vault_url": vault_url}, default=str)`, `service_detail="KeyVault"`.

2. `_scan_app_gateways(credential, subscription_id, logger)` — Uses `azure.mgmt.network.NetworkManagementClient`. Calls `application_gateways.list_all()`. For each gateway with ssl_policy: extracts `min_protocol_version`, `cipher_suites`, `policy_name`. Creates CryptoEndpoint per gateway: `protocol="AZURE"`, `host=gateway.id`, `port=443`, `tls_version=min_protocol_version`, `cloud_scan_json=json.dumps(ssl_policy_dict, default=str)`, `service_detail="AppGateway"`.

**Public function:**
```python
def scan_azure_targets(
    subscription_id: str,
    keyvault_urls: list[str] | None = None,
    logger=None,
) -> list[CryptoEndpoint]:
    if not AZURE_AVAILABLE:
        if logger:
            logger.v("azure SDK not installed — Azure scanning unavailable")
        return []
    credential = DefaultAzureCredential()
    results = []
    for vault_url in (keyvault_urls or []):
        results.extend(_scan_keyvault_keys(credential, vault_url, logger))
    if subscription_id:
        results.extend(_scan_app_gateways(credential, subscription_id, logger))
    return results
```

Wrap all Azure SDK calls in try/except. On `HttpResponseError` or `ResourceNotFoundError`, log and skip the resource (per pitfall 4 — Key Vault requires list+get permissions).
