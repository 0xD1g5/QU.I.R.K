**RSA key size from modulus n (base64url):**
```python
def _rsa_key_bits_from_n(n_b64url: str) -> int | None:
    try:
        padded = n_b64url.replace("-", "+").replace("_", "/")
        padded += "=" * (4 - len(padded) % 4)
        return len(base64.b64decode(padded)) * 8
    except Exception:
        return None
```

- `scan_container_image(image_ref: str, timeout: int = 120, logger=None) -> list[CryptoEndpoint]` — uses `shutil.which("syft")` guard. Runs `[exe, image_ref, "-o", "json"]` via `subprocess.run(capture_output=True, text=True, timeout=timeout)`. Parses JSON, iterates `artifacts`, filters by `artifact["name"].lower() in CRYPTO_LIB_ALLOWLIST`. Each match: `CryptoEndpoint(host=image_ref, port=0, protocol="CONTAINER", cipher_suite=artifact["name"], tls_version=artifact["version"], container_scan_json=json.dumps(artifact), scanned_at=datetime.now(timezone.utc).replace(tzinfo=None))`.
- `scan_container_targets(targets: list[str], timeout: int = 120, logger=None) -> list[CryptoEndpoint]` — iterates image refs, calls `scan_container_image` for each.

**quirk/scanner/source_scanner.py** — Clone the subprocess pattern per D-11, D-12, D-13, D-14:

- `scan_source_repo(repo_path: str, timeout: int = 300, logger=None) -> list[CryptoEndpoint]` — uses `shutil.which("semgrep")` guard. Runs `[exe, "--json", "--config", "p/cryptography", repo_path]` via `subprocess.run`. Parses JSON `results` array. Each finding: `CryptoEndpoint(host=repo_path, port=0, protocol="SOURCE", cipher_suite=result["check_id"], service_detail=f"{result['path']}:{result['start']['line']}", source_scan_json=json.dumps(result), scanned_at=...)`.

Both modules wrap all subprocess and JSON parsing in try/except, returning empty list on any failure.
