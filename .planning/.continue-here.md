<decisions_made>

- Use `localhost:20005` (not `registry:5000`) as push target — docker push goes through host daemon, not container network
- Use `python3 -m pip` instead of `pip3` on ubuntu:18.04 — the `pip3` shim bypasses `pip install --upgrade pip`

- `--no-build-isolation` required for `cryptography==2.9.2` — it predates PEP 517 isolated builds and needs global setuptools visible
